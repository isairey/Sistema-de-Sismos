# 🌍 Earthquake Information

Aplicación web desarrollada para la visualización y consulta de información sobre sismos en tiempo real, permitiendo a los usuarios acceder a datos relevantes de actividad sísmica de manera rápida e intuitiva.

---

## 📌 Descripción

**Earthquake Information** es una plataforma que muestra datos de terremotos obtenidos desde servicios externos (APIs), proporcionando información como magnitud, ubicación, fecha y profundidad.

El sistema está diseñado para facilitar la comprensión de la actividad sísmica mediante una interfaz clara y dinámica.

---

## 🚀 Características

- 🌍 Visualización de sismos en tiempo real
- 📍 Ubicación geográfica de eventos
- 📊 Información detallada (magnitud, fecha, profundidad)
- 🔍 Búsqueda y filtrado de datos
- ⚡ Actualización dinámica de información
- 🧾 Listado organizado de eventos sísmicos

---

## 🛠️ Tecnologías utilizadas

- **JavaScript (ES6+)** → Lógica de la aplicación  
- **HTML5** → Estructura  
- **CSS3** → Diseño y estilos  
- **API de sismos** → Fuente de datos
